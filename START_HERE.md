## ğŸ’¡ Pro Tips

### Development
- **Hot reload works** - Save files and see changes instantly
- **TypeScript strict mode** - Catch errors early
- **Shared packages** - Reuse code between projects
- **Independent deploys** - Each app deploys separately

### File Organization
- App-specific code goes in `apps/[app-name]/`
- Shared code goes in `packages/shared-*/`
- Documentation goes in `docs/` or root

### Workspace Commands
```bash
# Install package in specific app
npm install [package] -w apps/galion-initiative
npm install [package] -w apps/the-machine

# Run command in specific app
npm run [command] -w apps/galion-initiative
```
